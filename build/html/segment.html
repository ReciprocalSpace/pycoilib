draw(ax=None, draw_current=True) → None

Plot the segment in 3-dimension

This method draws the segment on an 3D-axis specified by ax, or a new figure otherwise. If the draw_current parameter is True, then the plot contains visual indicators of the the current orientation in the wire.

Parameters

  • ax (Axes) – Axis on which the segment is drawn. If no axis if provided, a new figure is created. Default is None.

  • draw_current (bool) – Specify if indicators are added to the figure to display the current orientation in the wire. Default is True.

get_vec_uvw(arc_rot_angle: float, axis_rot_angle: float, axis: numpy.ndarray) → Tuple[numpy.ndarray, numpy.ndarray, numpy.ndarray]

Return an orthonormal basis (u,v,w) = R_axis(axis_rot_angle) R_z(arc_rot_angle) (x,y,z)

This method transform the usual basis (x,y,z) in the arc segment reference frame into the laboratory frame (u,v,w). First, a rotation arc_rot_angle is applied around the z_axis (in the arc frame), and then a second rotation of axis_rot_angle around the specified axis is applied.

Parameters

  • arc_rot_angle (float) – Angular position of the arc around the z-axis, with the x-axis the origin.

  • axis_rot_angle (float) – Rotation angle of the second operation around the axis.

  • axis (numpy.ndarray) – Rotation axis for the second operation. 1D of shape (3,)

Returns

Arc usual basis represented in the laboratory frame.
